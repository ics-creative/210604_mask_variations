# Mask Variations

CSS、SVG、Canvasで作るマスク表現のサンプル集です。各ページは単体で開けるデモになっています。

[変幻自在なグラフィック表現！CSS, SVG, Canvasでマスクを使いこなせ - ICS MEDIA](https://ics.media/entry/210701/) のサンプルです。

## Run

`index.html` をブラウザで開きます。

## Demos

- CSS
  - `css_mask_image.html`: `mask-image`
  - `css_clip_path.html`: `clip-path`
  - `css_background_clip.html`: `background-clip: text`
- SVG・Canvas
  - `svg_sample.html`: SVGの `clipPath` / `mask`
  - `canvas_sample.html`: Canvasの `globalCompositeOperation`
- Practice
  - `practice_gradient_text.html`: グラデーションマスク
  - `practice_gradient_button.html`: グラデーションのアニメーション
  - `practice_radial_gradient.html`: `radial-gradient` マスク
  - `practice_sprite.html`: スプライトイメージとマスク
  - `practice_window_mask.html`: 背景に依存しないマスク
